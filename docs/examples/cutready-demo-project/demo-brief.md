# CutReady Demo Brief

This demo project is used by the documentation screenshot pass. It is intentionally
small, portable, and free of nested repository state.

## Story

CutReady helps demo creators plan first, then refine and validate with confidence.
The demo should emphasize:

- Sketch rows as the structured unit of demo thinking.
- Storyboards as the pacing layer.
- Draftline snapshots as safe versioning and collaboration.
- Auditaur drills as the real Tauri app confidence path.

## Presenter note

Do not over-explain recording. Recording and replay are later-phase capabilities;
the current demo should celebrate planning, AI refinement, assets, snapshots,
merge/conflict readiness, and release diagnostics.
