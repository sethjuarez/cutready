## Download

CutReady provides pre-built installers for all major platforms. Download the latest
release from [GitHub Releases](https://github.com/sethjuarez/cutready/releases/latest).

<Tabs>
  <TabItem label="Windows">
    Download the `.msi` or `.exe` (NSIS) installer from the release page:
    - **x64** (Intel/AMD): `CutReady_x.x.x_x64-setup.exe` or `.msi`
    - **ARM64** (Snapdragon / Copilot+ PCs): `CutReady_x.x.x_arm64-setup.exe` or `.msi`

    1. Run the installer
    2. Follow the setup wizard
    3. Launch CutReady from the Start Menu

    <Aside type="note">
      Windows builds are not yet code-signed. You may see a SmartScreen warning —
      click "More info" → "Run anyway" to proceed.
    </Aside>
  </TabItem>

  <TabItem label="macOS">
    Install with Homebrew:

    ```bash
    brew tap sethjuarez/cutready https://github.com/sethjuarez/cutready
    brew install --cask cutready
    ```

    To upgrade later:

    ```bash
    brew update
    brew upgrade --cask cutready
    ```

    Or download the `.dmg` file for your architecture:

    - **Apple Silicon** (M1/M2/M3): `CutReady_x.x.x_aarch64.dmg`
    - **Intel**: `CutReady_x.x.x_x64.dmg`

    1. Open the `.dmg` file
    2. Drag CutReady to your Applications folder
    3. Launch normally from Applications or Spotlight
  </TabItem>

  <TabItem label="Linux">
    Download the `.deb` package or `.AppImage`:

    **Debian/Ubuntu (.deb):**
    ```bash
    sudo dpkg -i cutready_x.x.x_amd64.deb
    ```

    **AppImage:**
    ```bash
    chmod +x CutReady_x.x.x_amd64.AppImage
    ./CutReady_x.x.x_amd64.AppImage
    ```

    <Aside type="tip">
      The Linux build requires PipeWire and GBM libraries for screen capture.
      On Ubuntu 24.04+, these are typically pre-installed.
    </Aside>
  </TabItem>
</Tabs>

## macOS Homebrew install

The recommended macOS install path is the CutReady Homebrew tap:

```bash
brew tap sethjuarez/cutready https://github.com/sethjuarez/cutready
brew install --cask cutready
```

This installs the notarized macOS build into `/Applications`. The cask chooses
the right download for your Mac automatically:

- **Apple Silicon** Macs use `CutReady_aarch64.dmg`
- **Intel** Macs use `CutReady_x64.dmg`

If you already installed CutReady from the `.dmg`, quit the app before installing
or upgrading with Homebrew.

### Troubleshooting Homebrew

If Homebrew cannot find the cask, refresh the tap and try again:

```bash
brew untap sethjuarez/cutready
brew tap sethjuarez/cutready https://github.com/sethjuarez/cutready
brew install --cask cutready
```

If the download fails with a missing asset, install the latest `.dmg` from
[GitHub Releases](https://github.com/sethjuarez/cutready/releases/latest). The
Homebrew cask depends on the latest release publishing both macOS assets:

- `CutReady_aarch64.dmg`
- `CutReady_x64.dmg`

To remove CutReady and its local app data:

```bash
brew uninstall --zap --cask cutready
```

## Prerequisites

CutReady uses **Playwright** under the hood for browser automation. For recording
features, you'll need:

- **Chrome** or **Microsoft Edge** installed
- An active browser profile (CutReady detects installed profiles automatically)

## Building from Source

If you prefer to build from source:

```bash
# Clone the repository
git clone https://github.com/sethjuarez/cutready.git
cd cutready

# Install frontend dependencies
npm install

# Run in development mode
npm run tauri dev

# Build for production
npm run tauri build
```

### Build Prerequisites

- **Node.js** (LTS)
- **Rust** (stable)
- **Tauri CLI v2** (`npm install -g @tauri-apps/cli`)

For Linux, you'll also need:
```bash
sudo apt-get install -y libwebkit2gtk-4.1-dev libappindicator3-dev \
  librsvg2-dev patchelf libpipewire-0.3-dev libgbm-dev
```
